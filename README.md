# ห้องเฝ้าทอง · Gold News Bot 🪙

บอทที่ดึงข่าวสำคัญที่ส่งผลต่อราคาทองคำ (สงคราม/ภูมิรัฐศาสตร์, Fed/ดอกเบี้ย, กองทุน SPDR & ธนาคารกลาง, ดอลลาร์) แล้วเด้งเข้า **Telegram** ของคุณ **อัตโนมัติทุก 1 ชั่วโมง** รันบน **GitHub Actions** ฟรี — ไม่ต้องเปิดคอมพิวเตอร์ทิ้งไว้

ส่งเฉพาะ "ข่าวใหม่" ที่ยังไม่เคยส่ง (จำว่าส่งอะไรไปแล้ว) จึงไม่รก ไม่ส่งซ้ำ

---

## ตั้งค่าครั้งเดียว (ประมาณ 10 นาที)

### 1) สร้างบอท Telegram และเอา Token
1. เปิด Telegram ค้นหา **@BotFather** แล้วกดเริ่ม
2. พิมพ์ `/newbot` ตั้งชื่อบอท → BotFather จะให้ **token** หน้าตาแบบ `123456789:AAE...` เก็บไว้

### 2) หา Chat ID ของคุณ
1. ค้นหาบอทที่เพิ่งสร้าง แล้วกด **Start** / ทักอะไรก็ได้ไปหาบอทหนึ่งครั้ง (สำคัญ ไม่งั้นบอทส่งหาคุณไม่ได้)
2. ค้นหา **@userinfobot** ใน Telegram แล้วกด Start → มันจะบอก **Id** ของคุณ (ตัวเลขล้วน) คือ chat id

### 3) อัปโหลดโค้ดขึ้น GitHub
1. สมัคร/เข้า GitHub สร้าง repository ใหม่ (เลือก Private ก็ได้)
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้นไป โดย**คงโครงสร้างเดิม**:
   ```
   gold_news_bot.py
   state/seen.json
   .github/workflows/gold-news.yml
   ```

### 4) ใส่ Token เป็น Secret (ปลอดภัย ไม่โผล่ในโค้ด)
ไปที่ repo → **Settings → Secrets and variables → Actions → New repository secret** เพิ่ม 2 ตัว:
| ชื่อ | ค่า |
|---|---|
| `TELEGRAM_TOKEN` | token จาก BotFather |
| `TELEGRAM_CHAT_ID` | ตัวเลข chat id ของคุณ |

### 5) เปิดใช้งานและทดสอบ
1. ไปแท็บ **Actions** → ถ้ามีปุ่มให้ enable workflows ให้กดเปิด
2. เลือก workflow **Gold News Bot** → กด **Run workflow** เพื่อทดสอบทันที
3. ภายในไม่กี่วินาทีควรได้ข้อความ "✅ ห้องเฝ้าทองออนไลน์แล้ว" + ข่าวล่าสุดเข้า Telegram

เสร็จแล้ว! จากนั้นมันจะรันเองทุกชั่วโมงโดยอัตโนมัติ

## ปรับแต่งได้

- **ความถี่:** แก้บรรทัด `cron` ในไฟล์ `.github/workflows/gold-news.yml`
  - ทุก 30 นาที: `'*/30 * * * *'` · ทุก 2 ชั่วโมง: `'0 */2 * * *'`
- **ภาษาเป็นข่าวไทย:** ในไฟล์ workflow เปลี่ยน `LANG_HL: 'en-US'` เป็น `LANG_HL: 'th'`
- **หัวข้อข่าวที่ตาม:** แก้ลิสต์ `QUERIES` ในไฟล์ `gold_news_bot.py` (เพิ่มน้ำมัน, บอนด์ยีลด์ ฯลฯ ได้)
- **จำนวนข่าวต่อรอบ:** แก้ `MAX_PER_RUN` ในไฟล์ workflow

## แหล่งข่าว
ตัวดึงข่าวใช้ **Google News RSS** เป็นตัวกลาง (index ข่าวด่วนไว เสถียร ไม่โดนบล็อก) แล้ว**กรองให้เหลือเฉพาะสำนักข่าวใหญ่ต่างประเทศ**ที่ออกข่าวไว ตามลิสต์ `ALLOWED_SOURCES` ใน `gold_news_bot.py` ปัจจุบันคือ:

> Reuters · Bloomberg · CNBC · Financial Times · Wall Street Journal · Barron's · MarketWatch · FXStreet · Investing.com · Kitco · DailyFX · Associated Press · AFP · Yahoo Finance · ForexLive · Mining.com

- อยากเพิ่ม/ลดสำนัก: แก้ลิสต์ `ALLOWED_SOURCES` (พิมพ์ชื่อเป็นตัวเล็ก ขอแค่บางส่วนของชื่อก็พอ เช่น `"reuters"`)
- อยากรับ**ทุกแหล่ง**ไม่กรอง: ตั้ง `STRICT_SOURCES: '0'` ในไฟล์ workflow

## ทดสอบบนเครื่องตัวเองก่อนได้ (ไม่บังคับ)
```bash
pip install requests feedparser
python gold_news_bot.py        # ไม่ใส่ token = โหมดทดสอบ พิมพ์ข่าวออกจอเฉยๆ ไม่ส่งจริง
```

## ข้อควรรู้
- ตารางเวลาของ GitHub Actions อาจ**ดีเลย์ 5–15 นาที**ในช่วงระบบโหลดสูง ถือเป็นเรื่องปกติ
- ถ้า repo **ไม่มีกิจกรรมเกิน 60 วัน** GitHub จะปิด scheduled workflow ชั่วคราว แค่เข้าไปกด enable หรือ push อะไรสักครั้งก็กลับมาทำงาน
- ฟรีภายใต้โควต้า GitHub Actions (public repo ฟรีไม่จำกัด · private repo มีโควต้าฟรี 2,000 นาที/เดือน งานนี้ใช้ราว 1 นาที/รอบ ≈ ใต้โควต้า)
- บอทจะ commit ไฟล์ `state/seen.json` กลับเข้า repo ทุกรอบ เพื่อจำว่าส่งข่าวอะไรไปแล้ว — เป็นพฤติกรรมปกติ

> ⚠️ ป้ายหมวดหมู่เป็นการจัดกลุ่มข่าวตามคีย์เวิร์ดเท่านั้น บอทตั้งใจ**ไม่ชี้ขึ้น/ลงหรือสัญญาณซื้อขาย** ใช้เป็นข้อมูลประกอบ การตัดสินใจเทรดให้ยึดกราฟและการวิเคราะห์ของคุณเอง
